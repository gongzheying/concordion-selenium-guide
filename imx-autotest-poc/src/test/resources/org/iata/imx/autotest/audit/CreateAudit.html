Create a audit

  1. Navigate to page "New Audit" in AUDIT/INSPECTIONS module, the "New Audit" page will be displayed.
  2. Enter or select the value as the following table, and Click the Save button, the "View Audit" page should be displayed.
    Type Subtype Location Auditee Scheduled Start Date Scheduled End Date Lead Auditor Scope
    Internal IOSA Self Assessment AAA Administrative Today Tomorrow Administrator Imx Process >> Administrative
  3. Click the Save button, the alarm information "The changes have been saved!" is displayed.
